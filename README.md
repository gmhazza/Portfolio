# Portfolio

![Logo Image](./public/logo.jpg)

This Website showcasing m skills on web designing in Vue3 and Tailwindcss

## Project Setup

```sh
git clone https://github.com/gmhazza/Portfolio.git
cd Portfolio
npm install
```

### Compile for Development

```sh
npm run dev
```

### Compile for Production

```sh
npm run build
```
